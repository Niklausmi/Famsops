User Management
Admin only — manage CRM access via Supabase Auth
ℹ How user management works with Supabase Users are authenticated by Supabase Auth. Each user has an email address + password. Their role, name and display settings are stored in the profiles table. To add a user: click Invite User, fill the form — the system creates their Supabase Auth account and profile in one step using your service key.
⚠

Service Key Required for User Management

Creating users requires the Supabase service_role key (not the anon key). Follow these steps:

1Go to Supabase Dashboard → Project Settings → API
2Copy the service_role key (keep secret — never commit to git)
3Open crm-core.js and add: const SUPABASE_SERVICE_KEY = 'your-service-key';
4Reload this page
